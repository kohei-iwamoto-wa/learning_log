#### AWS Price List API 

AWS Price List APIは、AWSが提供する各サービスやリソースの最新の価格、および製品属性（SKU情報、インスタンスタイプ、OSなど）をプログラムから自動的に取得するための仕組みです。

#### AWS CUR(AWS Cost & Usage Reports)

AWSの利用状況と利用料金を最も細かく最も包括的なレポート

* S3 にレポートファイルを配信する
* レポートを１日３下位まで更新する
* CUR API AWS リファレンスを使用してレポート作成、取得削除する
