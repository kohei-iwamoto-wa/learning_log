### tfstate ファイルをS3 で管理する方法

backend.tf ファイルを S3 で管理するには？

backend.tf ファイルを作成し、以下の設定を記載する。
```
terraform {
  backend "s3" {
    bucket         = "your-bucket"
    key            = "path/to/terraform.tfstate"
    region         = "us-west-2"
    encrypt        = true
    use_lockfile = true
  }
}
```

backend.tf を記載したあと、以下のコマンドを順番に実施する

terraform init

terraform apply
※ このタイミングでS3に反映される
