### 学んだこと

cognito のグループは、アクセス権限の管理や制御を行うことができます。

例えば、画面の表示、非表示を制御に利用する。

```
ログイン
   ↓
Cognito
   ↓
JWT発行（cognito:groups = Admin）
   ↓
フロント
   ├─ ボタン表示・非表示
   └─ API呼び出し
             ↓
API Gateway
             ↓
Lambda
   ├─ JWT検証
   ├─ グループ確認
   └─ OKなら処理、NGなら403

```
