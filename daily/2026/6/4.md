### 学んだこと

#### AWS Application Discovery Service

AWS Application Discovery Serviceは、オンプレミスサーバーのデータ（CPU/メモリの使用率、動作しているプロセス、サーバー間のネットワーク依存関係など）を自動で収集し、AWSへの移行計画（マイグレーション）を支援するためのサービスです。

* できること
  * サーバー間の「依存関係」の可視化
    * サーバー間の通信を可視化できる
  * 収集したCPU等の利用実績をベースに、AWS移行時にどのEC2インスタンスを選べばコストパフォーマンスが最適になるか提案する

#### AWS Migration Hub

AWS Application Discovery Service で収集されたデータは自動的に「AWS Migration Hub」に集約される。

#### AWS Application Migration Service

オンプレミスの物理サーバーや仮想サーバー（VMware、Hyper-V）、または他のクラウドにあるサーバーを、自動かつ安全にAWS（Amazon EC2）へ移行（マイグレーション）するための主要なサービス
