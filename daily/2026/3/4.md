### Helm を利用したバージョンアップ方法について


1. リリースの一覧を取得する。

まずは、アップグレード対象のリリースが存在する namespace と、現在のステータスを確認します。

```
helm ls -n <namespace> 
```

2. リリースの値を書く出す(バックアップ)

```
helm get values <RELEASE_NAME> -n <namespace> -o yaml > old_values.yaml
```

3. ドライラン

```
helm upgrade <RELEASE_NAME> <CHART_NAME> \
  -n <namespace> \
  -f old_values.yaml \
  -f override-demo.yaml \
  --dry-run
```

4. アップグレードの実行

```
helm upgrade <RELEASE_NAME> harness/harness \
  -n <namespace> \
  -f old_values.yaml \
  -f override-demo.yaml \
  --wait \
  --history-max 5
```
