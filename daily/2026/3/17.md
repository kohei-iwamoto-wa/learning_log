### lsof (List Open File)

#### lsof とは

lsof コマンドは、Linux 上の開かれているファイルを一覧するコマンドです。

Linux では「すべてはファイルである（ネットワークソケットやデバイスも含む）」という哲学があるため、このコマンド一つでプロセス、ポート、ファイルの使用状況を確認することができます。

※ 「すべてはファイルである」について理解できていない。。

#### 実行例

443 ポートを専有しているプロセスを表示する

```
sudo lsof -i :443
```

ファイルを開いているユーザーを表示する。

```
sudo lsof -u username
```

### ソケットファイル

ソケットファイルは、同一マシン内のプロセス間通信を行うために用いられる特殊なファイルです。

同一マシン上のメモリを介してデータの受け渡しをするため、通信を高速に行うことができます。

※ ネットワーク階層の処理を行う必要がないため。


####　ソケットファイルの例

Docker コマンドとDocker デーモンの通信で利用されています。

```
/var/run/docker.sock
```

#### なぜファイル形式か？

権限管理：権限管理を通常のファイルのようにchomod を利用することができ設定が楽

名前指定： IP:PORT の形式ではリーダブルではないが、docker.sock のようにするとリーダブルでわかりやすくなる
