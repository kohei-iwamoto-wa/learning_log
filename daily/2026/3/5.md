## 2026-03-05 学んだこと

### cert-manager

cert-manager、KubernetesまたはOpenShiftクラスター内のTLC証明書作成、有効期限が切れる前に更新する。

Let's Encryptなど様々な証明機関から証明書を取得することができる。


### EKSのpodが起動しないときに利用するコマンド

```
# podが立ち上がらない理由が書いてある。
kubectl describe pod <Pod名>
```

```
# Helmを利用している場合前のバージョンに戻せる
helm rollback <リリース名> <リビジョン番号> -n <名前空間>
```
