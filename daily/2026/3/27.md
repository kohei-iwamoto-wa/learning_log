### ingress

#### ingress とは

クラスター内のサービスへの外部アクセスを管理する API オブジェクトです。

クラスター外部から　HTTP、HTTPS ルートをクラス内に公開する。トラフィックのルーティングは、Ingress で定義されたルールによって制御する。 

※ Ingressを満たすには、 AWS Application LoadBalancer Controller などの Ingress Controller が必要です。

#### 追加機能開発は行われない

追加機能開発は行われないようです。

公式の推奨は、Ingress の代わりに Gateway を利用すること

ただし、Ingress API の安全性は保証され削除される予定はない。
