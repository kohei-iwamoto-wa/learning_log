### Helmとは

Helmは、Kubernetes用のバッケージマネージャーです。

### Helmインストール

```
# Helmのインストーラスクリプトの取得
$ curl -fsSL -o get_helm.sh https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3
$ chmod 700 get_helm.sh
$ ./get_helm.sh

# インストールの確認
/usr/local/bin/helm version
# シンボリックリンク（一時対応）
sudo ln -s /usr/local/bin/helm /usr/bin/helm
version.BuildInfo{Version:"v3.20.0", GitCommit:"b2e4314fa0f229a1de7b4c981273f61d69ee5a59", GitTreeState:"clean", GoVersion:"go1.25.6"}
```

### aws-load-balancer-controllerのインストール

```
sh-4.2$ helm repo add eks https://aws.github.io/eks-charts
sh-4.2$ helm repo update

# aws-load-balancer-controller のインストール
helm install aws-load-balancer-controller eks/aws-load-balancer-controller \
  -n kube-system \
  --set clusterName=eks-work-cluster \
  --set serviceAccount.create=false \
  --set serviceAccount.name=aws-load-balancer-controller \
  --set vpcId=vpc-xxxxxxxxxxxxxxxxx \
  --set region=ap-northeast-1 \
  --version 1.11.0

※--set fargateTypeMeta.enabled=true #Fargateの設定

AWS Load Balancer controller installed!

# aws-load-balancer-controller稼働確認
sh-4.2$ kubectl get deployment -n kube-system aws-load-balancer-controller

```


### aws-load-balancer-controllerのバージョンアップ

```
helm repo update eks
# CRDは自動バージョンアップ
kubectl apply -k "github.com/aws/eks-charts/stable/aws-load-balancer-controller/crds?ref=master"

```

### Helmのコマンド

```
# 現在登録されているリポジトリの一覧を表示
sh-4.2$ helm repo list
NAME    URL
eks     https://aws.github.io/eks-charts

# 登録したすべてのリポジトリから最新のリストを取得する
helm repo update

# helmでインストールされた
sh-4.2$ helm list -n kube-system
NAME                            NAMESPACE       REVISION        UPDATED        STATUS          CHART                                   APP VERSION
aws-load-balancer-controller    kube-system     1               2026-03-03 12:48:13.710298763 +0000 UTC      deployed        aws-load-balancer-controller-3.1.0      v3.1.0
```

### eksctlでバージョン確認

```
kubectl get deployment -n kube-system aws-load-balancer-controller \
  -o jsonpath='{.spec.template.spec.containers[0].image}'
```
