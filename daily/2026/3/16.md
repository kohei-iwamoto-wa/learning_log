### systemctl

systemctl とは、systemd を操作するためのコマンドでサービスの起動、停止の設定などを行います。

| 操作 | 例 |
| ---- | ---- |
| 起動 | systemctl start nginx |
| 停止 | systemctl stop nginx |
| 再起動 | systemctl restart nginx |
| 状態確認 | systemctl status nginx |
| 自動起動設定 | systemctl enable nginx |
| 自動起動解除 | systemctl disable nginx |
