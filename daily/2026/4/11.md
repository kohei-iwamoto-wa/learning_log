#### curl -v 

curl -v https://www.google.com > /dev/null\

curl -v で http request や http response のヘッダーの詳細を表示することができます。

出力記号の意味
* (アスタリスク): 接続プロセス（DNS解決、TCP接続、TLSハンドシェイク）。

> (右向き不等号): あなたのPCからサーバーへ送った Request Header。

< (左向き不等号): サーバーから返ってきた Response Header。

ステップアップ：特定のリクエストを模倣する

テスト時に使用する

User-Agentを偽装する:
curl -v -H "User-Agent: MyCustomBot/1.0" https://...

Hostヘッダーを書き換える（ALBやWAFの検証で多用）:

curl -v -H "Host: example.com" http://[IPアドレス]
