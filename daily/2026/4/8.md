#### Cost and Usage Report

包括的コストレポート

管理アカウント権限

S3統合とデータ形式

#### できること

* Amazon S3 バケットにレポートファイルを配信する。
* レポートを 1 日に 3 回まで更新する。
* CUR API AWS リファレンスを使用してレポートを作成、取得、削除する。
* QuickSight や Athena などと連携した。

* 大量のAWSアカウントのコスト可視化
  * AWS Organizations管理アカウントからAWS Cost and Usage Report（CUR）を作成する。各チームがAmazon QuickSightダッシュボードを通じてCURを可視化できるようにする。

#### 学んだこと

AWS Cost and Usage Report で管理アカウントの各メンバアカウントのコスト、利用状況を可視化することができます。

#### Lambda  デプロイ　自動ロールバック

便利そうな機能やな

```
AWS SAM (Serverless Application Model): * AutoPublishAlias によるバージョン管理とエイリアスの自動生成。

AWS CodeDeploy (Lambda):

デプロイ設定: Canary（10%だけ流す）や Linear（徐々に増やす）の違いを理解。

ライフサイクルフック: BeforeAllowTraffic と AfterAllowTraffic でテスト用Lambdaを動かす実装。

CloudWatch アラームとの連携:

エラー率（5xxエラー）やレイテンシを監視し、CodeDeployに自動ロールバックさせる設定。

テストに失敗やエラー率上昇した時はロールバック
```
