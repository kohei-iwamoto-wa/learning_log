### マルチアーキテクチャビルドの方法

マルチアーキテクチャビルドには２つの方式がある

docker buildx を利用する

#### エミュレート

ARM で AMDのイメージをビルドする方式

ARM から AMD に翻訳するのでビルドスピードは落ちます。

#### ビルドクラウド

Docker が提供するクラウド環境で実機を利用してビルドすることができます。

翻訳作業がいらないのでビルドスピード早い

#### コマンド

```
$ docker buildx build \   --builder=cloud-nigelpoulton-ddd-cloud \   --platform=linux/amd64,linux/arm64 \   -t nigelpoulton/tu-demo:latest --push .
```
