#### aws s3 sync

aws s3 sync は、ローカルからS3バケットやS3バケットからS3バケットへデータを同期するためのコマンドです

同期の条件は、ローカルからS3バケットへのパターンの場合、ローカルファイルの最終更新時刻がS3オブジェクトよりも新しい時刻であり、かつ、ローカルファイルが指定されたバケットとプレフィックスの下に存在しない場合、同期が行われます。

```
# ローカルからS3バケットへ
aws s3 sync . s3://amzn-s3-demo-bucket

# ログ出力
upload: test.txt to s3://amzn-s3-demo-bucket/test.txt
upload: test2.txt to s3://amzn-s3-demo-bucket/test2.txt
```

#### S3 アクセスポイント

従来のS3バケットポリシーは、バケットに対して１つのポリシーのみ定義可能でした。

しかしながら、S3 アクセスポイントの登場でアクセスポイントごとでポリシー設定可能になり、細かく設定を切り分けることができるようになった。

１つのポリシーに書き込まなくていいので、バケットポリシーの肥大化を防ぐことができる
