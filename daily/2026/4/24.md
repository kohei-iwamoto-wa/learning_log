### 再起動ポリシー

コンテナの状態とポリシーの組み合わせで再起動するか決まっている

| ポリシー | 異常終了 | 正常終了 | docker stop 実行 | デーモン再起動時 |
| :--- | :--- | :--- | :--- | :--- |
| **no** | N | N | N | N |
| **on-failure(失敗時のみ)** | Y | N | N | Y |
| **always(常に)** | Y | Y | N | Y |
| **unless-stopped(停止されない限り)** | Y | Y | N | N |

### 再起動ポリシーを指定する

docker run --name neversaydie -it --restart always alpine sh
